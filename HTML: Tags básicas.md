# Introdução

Nesse arquivo vamos aprender a usar as tags básicas do html:

  - ```<title></title>``` - Exibe o título de uma página web
    
  - ```<h1></h1> ... <h6></h6>``` - criar títulos
    
  - ```<p></p>``` - criar parágrafos
    
  - ```<a></a>``` - criar um link
    
  - ```<img>``` - exibir uma imagem
    
  - ```<div></div>``` - agrupar elementos

## Conceitos

- Tag ```<title><title>``` - Permite que o usuário identifique o tipo de conteúdo que ele pode esperar dentro daquela página e tem sua importância para os mecanismos de busca, como o Google. Portando precisa ser relevante e relacionado ao conteúdo da pagina
